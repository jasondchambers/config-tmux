## config-tmux

Leader key is C-b

| Key | Binding |
|-----|---------|
| \<leader> r | reload conf |
| \<leader> \| | new vertical pane |
| \<leader> - | new horizontal pane |
| \<leader> j,k,l,h | resize pane |
| \<leader> m | toggle maximize pane |
| C-h,j,k,l | navigate panes (consistent with Neovim) |
